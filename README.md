# GraphQL Authentication & Authorization

> GraphQL-2Auth

```cmd
# Install dependencies
$ yarn install

# Run project
$ yarn start

# Run project as dev with file watch
$ yarn dev

# Trigger test
$ yarn test
```

1. GraphQL API
1. Authentication & Authorization using JWT Token
1. Test using jest
